# PRD: Marie Cuisine 

## Project Title
**Marie Cuisine**

## One-Sentence Goal
A warm, timeless recipe website offering approachable, delicious recipes for all skill levels.

## Core Features
- **Recipe Library:** Users can browse and filter recipes by category (breakfast, lunch, dinner, desserts, healthy, quick meals).  
- **Recipe Pages:** Include ingredients, step-by-step instructions, prep time, difficulty, and photos.  
- **Search Bar:** Allows users to search recipes by name or ingredient.  
- **Featured Seasonal Recipes:** Highlighted on the homepage (e.g., fall dishes, summer grilling, holiday baking).  
- **User Favorites:** Basic version with a simple “save” button stored locally in the browser.  

## Intended Pages (4–6)
1. **Homepage:** Hero banner, featured recipes, categories.  
2. **Recipes:** Grid view with filters.  
3. **Individual Recipe Page:** Ingredients, instructions, photo, save button.  
4. **About:** Story of the site, mission, inspiration (Martha-Stewart-inspired).  
5. **Contact:** Simple form or email link.  
6. **(Optional) Tips & Guides:** Kitchen basics, pantry essentials, cooking techniques.  

## Design / Interaction Notes
- Clean, cozy aesthetic — white background, soft pastels, large food photography.  
- Rounded cards, friendly typography (serif for headers, sans-serif for body).  
- Sticky top navigation bar for quick access to recipes.  
- Optional dark mode toggle.
